# Examples

Put your own reference images in this folder.

This repository does not include a reference image by default because image rights and likeness rights depend on your source material.

Example command:

```bash
npm run video -- --prompt-file prompts/worldcup-fancam.example.txt --reference-image examples/your-storyboard.png --duration 10 --aspect-ratio 1:1 --resolution 720p --prefix worldcup-fancam
```
